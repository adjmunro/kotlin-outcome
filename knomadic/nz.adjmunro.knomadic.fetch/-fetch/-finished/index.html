Finished

@JvmInline
value class Finished<out T : Any>(val result: T) : Fetch<T>

A fetch operation that has finished.

Following the single-responsibility principle, the success or failure of the fetch is left up to the encapsulated type, T. It is recommended to use a Outcome type for this purpose.
